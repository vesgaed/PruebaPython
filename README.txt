

En este archivo proyecto se encuentra un codigo que permite encontrar la cantidad de dias habiles entre un rango de fechas. Se entiende que dias habiles son de lunes a viernes. Sabados, domingos y festivos no son dias habiles.

El programa solo calcula dias habiles entre las fechas 1 de agosto del 2020 y 31 de agosto del 2021. Si se quiere probar el porgrama sin limitantes, elimine el archivo 'limites' dentro de la carpeta 'configuracion'. 
Esta aplicacion utiliza una conexion a una API web, ademas de gestionar la respuesta y realizar el calculo respectivo. Todo se hizo con el fin de demostrar el conocimiento evaluado de progrmacion orientada a objetos, persistencia, manejo de excepciones y ademas, uso de arquitectura REST.

No utilice la libreria Holidays porque:
Queria poner en muestra el uso de servicios REST.
La cantidad de dias recibidos desde las funciones de la libreria, eran erroneos.
Queria usar la menor cantidad de librerias poco comunes.
